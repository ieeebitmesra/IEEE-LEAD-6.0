# CIFAR-10 Image Classification using Convolutional Neural Networks (CNN)

This project trains a deep Convolutional Neural Network (CNN) on the CIFAR-10 dataset using TensorFlow/Keras.  
It includes data preprocessing, augmentation, model development, training, evaluation, and visualization.


## Files
- `notebook.ipynb` — analysis and training steps
- `train.py` — training script
- `requirements.txt` — pip dependencies

## Results
- Accuracy: 0.92
- Loss: 0.29

## How to run
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
# open the notebook or run the script
