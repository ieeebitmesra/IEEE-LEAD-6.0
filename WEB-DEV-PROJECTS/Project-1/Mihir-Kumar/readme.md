# Portfolio Project

This portfolio project is a Flask based website showcasing my skills, projects and achievements.

## Demo
- Live demo: https://m141r.pythonanywhere.com/

## Files
- `templates` - HTML Files
- `static` - CSS File , SVG Files , Project Screenshot
- `app.py` - The main Flask application file that handles routing and serves the portfolio website.
- `readme.md` - This File

## How to run locally
```bash
python run app.py  # then open http://localhost:5000
```
